Requirements for Fake Certificate Detection Using Deep Learning
Overview

This project detects fake certificates using deep learning techniques. The system analyzes certificate images and identifies whether they are genuine or forged by learning visual patterns such as fonts, logos, seals, and layout structures. The model uses MobileNetV2 to classify certificates as real or fake.
