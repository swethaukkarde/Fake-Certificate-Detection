Setup Instructions

Follow the steps below to run the Fake Certificate Detection project.

1. Install Python
Make sure Python 3.x is installed on your system.

2. Clone the Repository

git clone https://github.com/your-username/Fake-Certificate-Detection.git

3. Navigate to Project Folder

cd Fake-Certificate-Detection

4. Install Required Libraries

pip install -r requirements.txt

5. Run the Project

python src/mobilenet.py

6. Upload Certificate

Upload a certificate image (JPG/PNG/PDF) when prompted to check whether it is **Real or Fake**.
